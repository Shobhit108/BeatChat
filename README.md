# 🎵 BeatChat

BeatChat is a full-stack real-time chat platform that combines messaging, music streaming, and AI-powered interactions into a single experience.

Built to make conversations more engaging through live messaging, music sharing, personalized themes, and smart interactions.

## Live Demo

🌐 Live: https://beatchat-yh6i.onrender.com

## Repository

💻 GitHub: https://github.com/Shobhit108/BeatChat

## Features

### Authentication & Security
- Secure User Authentication
- Protected Routes
- Session-Based User Experience

### Real-Time Communication
- Real-Time Messaging using Socket.IO
- Typing Indicators
- Online / Offline Presence Tracking
- Instant Message Synchronization

### Music Experience
- Search & Stream 90M+ Songs
- Deezer API Integration
- Music-Enhanced Conversations

### AI Features
- AI Chat Assistant (Neuron)
- Smart Interactive Experience

### Customization
- 32 Custom Themes
- Personalized Chat Experience

### Media Optimization
- Optimized Image Handling
- Faster Media Performance
- Responsive UI Across Devices

## Installation

Clone the repository

```bash
git clone https://github.com/Shobhit108/BeatChat.git
```

Go to project directory

```bash
cd BeatChat
```

Install dependencies

```bash
npm install
```

Run frontend

```bash
npm run dev
```

Run backend

```bash
npm run dev
```
